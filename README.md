# The Lehmann–Scheffé Theorem in Statistical Theory: Origin, Framework, and Modern Relevance

The paper discusses the framework of the Lehmann-Scheffé theorem, the conditions that it requires, a historical background on its development, applications, and its modern relevance.

## Contents
- `paper/lehmann_scheffe.pdf`: Full paper
- `paper/latex`: LaTeX code for paper
- `code/lehmann_scheffe_figures.R`: R code for the variance plot

## Context
Final project for STAT 411 (Statistical Theory), University of Illinois Chicago.
